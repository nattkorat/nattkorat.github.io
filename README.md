# Natt Korat

Personal academic website for Natt Korat, Lecturer-Researcher at the Cambodia
Academy of Digital Technology.

The site highlights research work in Khmer automatic speech recognition, speech
data processing, multilingual NLP, information extraction, and low-resource
language technologies.

## Files

- `index.html` - main one-page research profile
- `files/natt_korat_cv.pdf` - curriculum vitae PDF linked from the site
- `images/` - profile imagery used by the homepage
